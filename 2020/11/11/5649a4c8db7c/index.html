提交版本

git add 提交修改的文件到暂存区

git commit 提交暂存区的文件到当前分支

git status 查看暂存区是否存在修改未提交

git diff 查看暂存区文件的修改记录

git log 查看历史提交记录

回退版本

git 中 HEAD 表示当前版本，上一个版本就是 HEAD^，上5个版本就是 HEAD^5

zsh 中 HEAD^ 会失效，需要使用 HEAD\^ GITHUB ISSUE

git reflog 记录 git 操作历史

撤销修改

git checkout – 将文件状态恢复到上次提交到暂存区的状态

git reset HEAD 将文件状态恢复到上次提交到版本库的状态
